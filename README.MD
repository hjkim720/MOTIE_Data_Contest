# 제 13회 산업통상자원부 공공데이터 활용 아이디어 공모전
- 데이터 분석 부문 **대상(산업통상자원부 장관상) 수상**
- 한국중부발전의 태양광 발전이나 가스터빈 운전시 대기 상태에 따라 운전 여유도 및 연소 안정도를 예측 할 수 있도록 익일 24시간의 정확한 기상 예측이 필요, 관측 지점과 예측 지점의 지리적 이격으로 인한 기상 예측 오차를 보정하는 모델 개발
- 기존 대비 오차를 평균 51% 감소 시킴, 특히 대기압은 93% 감소 시킴
